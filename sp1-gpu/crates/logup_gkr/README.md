# sp1-gpu-logup-gkr

GPU-accelerated LogUp-GKR protocol implementation.

Implements the LogUp-GKR protocol on GPU for efficiently proving lookup table constraints. This protocol enables succinct verification of table lookups in SP1's constraint system.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
