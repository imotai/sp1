# sp1-gpu-challenger

GPU challenger state management for Fiat-Shamir transformations.

Manages the challenger state used in the Fiat-Shamir heuristic, providing the randomness needed to make interactive proofs non-interactive in the GPU proving pipeline.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
