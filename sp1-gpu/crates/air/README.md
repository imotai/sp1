# sp1-gpu-air

AIR (Algebraic Intermediate Representation) interface for SP1 machines on GPU.

Provides the GPU-compatible AIR definitions and constraint evaluation logic used by the SP1 zkVM. This crate bridges the SP1 machine definitions with the GPU prover infrastructure.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
