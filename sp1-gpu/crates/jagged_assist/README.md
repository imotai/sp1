# sp1-gpu-jagged-assist

GPU helper kernels for jagged polynomial operations.

Provides CUDA kernels that assist with jagged (variable-sized) polynomial operations, supporting the sparse-to-dense conversion process in the GPU proving pipeline.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
