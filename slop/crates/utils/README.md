# slop-utils

Utility functions for SLOP.

Built on [`p3_util`](https://crates.io/crates/p3_util) from [Plonky3](https://github.com/Plonky3/Plonky3), with additional functionality:

- Logger setup and configuration for SLOP crates
- Common utility functions from `p3_util`: `log2_ceil_usize, log2_strict_usize, reverse_bits_len`.
